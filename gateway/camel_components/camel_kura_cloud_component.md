# Camel Kura Cloud component

The Kura platform implements CloudService. CloudService provides some features to push and receive messages from outside. Camel Kura Cloud component interacts directly with CloudService.
Configuration uses the same convention than [CloudService](http://download.eclipse.org/kura/releases/1.3.0/docs/apidocs/org/eclipse/kura/cloud/CloudService.html) and [CloudClient](http://download.eclipse.org/kura/releases/1.3.0/docs/apidocs/org/eclipse/kura/cloud/CloudClient.html) from Kura API
